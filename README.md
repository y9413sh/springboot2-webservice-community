# 주요 기능
- 소셜 로그인 (OAuth 2.0)
    - Google
    - Naver
- 게시글 조회, 수정, 삭제
- 게시글 페이징 및 검색
- 댓글 (준비중)


 
# 개발 환경
- Java8
- IDE
    - IntelliJ
- Gradle
    - Gradle 6.0.1
- Spring Boot
    - v2.1.9.RELEASE
- template
    - mustache
- JPA
- REST API
- DB
    - h2
- JavaScript / Jquery / Ajax
- Css
  - BootStrap 
  - Google Font
